Use PopPUNK directly from the command line.

Command line

Advantages

  • All options available.
  • As flexible as possible.
  • Full documentation available.
  • Use any available mode, fit a new model or use an existing one.
  • Easy to install using conda.
  • Fastest most scalable mode, supports GPUs.

Disadvantages

  • Requires some Unix/command line knowledge.
  • Read documentation to understand usage options and parameters.

Install on a Galaxy server

Graphical interface and integrated data access to create new databases.

Galaxy wrapper

Available features

  • Use a graphical interface to run PopPUNK.
  • Upload your data via Galaxy.
  • All command line options are available.

Not yet available

  • Only database creation and model fit supported.
  • Assign query mode not yet implemented.
  • Read documentation to understand usage options and parameters.
  • Difficult to re-use intermediate outputs without re-running.
